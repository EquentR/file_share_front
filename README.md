# 基于Vue+CHFS的文件管理前端

## 1、基本使用：

### 安装项目
```
npm install
```

### 调试
```
npm run serve
```

### 打包
```
npm run build
```

### 修复
```
npm run lint
```

## 2、介绍

本前端基于Vue开发，后端采用成熟的CHFS（Cute Http File System）对接。只需要配置服务器请求地址，并将打包好的三大件部署在Nginx中即可使用！
